# 🧠 Brainy Fuse

> **An AI-Powered Productivity & Knowledge Management Platform**

Brainy Fuse is an intelligent platform that combines Artificial Intelligence, real-time collaboration, secure authentication, and cloud-based services into a single application. It is designed to help users organize information, automate repetitive tasks, communicate efficiently, and interact with AI through a modern and scalable architecture.

The goal of Brainy Fuse is to build an ecosystem where AI assists users in learning, managing knowledge, generating content, and improving productivity.

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Brainy-Fuse.git
```

Navigate to the project directory:

```bash
cd Brainy-Fuse
```

Run the application:

```bash
mvn spring-boot:run
```

---

# ⚙️ Configuration

Configure your application properties:

```properties
spring.datasource.url=
spring.datasource.username=
spring.datasource.password=

jwt.secret=

google.client.id=
google.client.secret=

gmail.api.key=
```
